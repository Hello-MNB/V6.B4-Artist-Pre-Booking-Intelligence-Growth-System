- **Public Passport split into two buyer personas** — Booking view (event booker) + Representation view (agency). Separate files, one shared firewall-safe data layer; hero toggle + `?view=rep` deep link. (`270dde8`)

- **PWA stale-cache trap** — self-destroying service worker un-traps users stuck on old cached app. (`6205ad2`)
